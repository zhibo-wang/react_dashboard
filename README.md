react练手项目

项目说明
    该项目使用create-react-app脚手架搭建，用于个人学习react和ant design的使用

项目运行
    1.clone项目
    2.npm install
    3.npm start
目录结构
    root-node_modules
        -public
        -src
            -layout
            -App.css
            -App.js
            -index.css
            -index.js
            -Login.js //登录组件
        -package.json
        -yarn.lock
